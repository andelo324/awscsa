Compute

EC2
Elastic Compute - Virtual Machines
EC2 Container Services
a container management service that suports Docker containers
Elastic Beanstalk
orchestration service that automatically handles the details of capacity provisioning, load balancing, scaling and application health monitoring
Lambda
It is a compute service that runs code in response to events and automatically manages the compute resources required by that code
Lightsail
Amazon's VPS service

Storage

S3
known as Simple Service Storage - object based storage (buckets)
EFS
known as Elastic File service - esentially network attached storage where you can mount this to multiple machines
Glacier
archival storage
Snowball
Way to bring in large amounts of data to AWS using physical disk and ship it to AWS
Storage Gateway
Essentially virtual appliances you install in your datacenter or head office and then will replicate data back to S3. 4 Different types of Storage Gateway
